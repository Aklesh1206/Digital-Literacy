##  Cybersecurity: Stay Safe Online
One of my core takeaways from this project is the **"Literate Engineer's Code"**:
1. **Never** share a UPI PIN to *receive* money.
2. **Always** enable 2FA on GitHub and Google.
3. **Report** fraud immediately at **1930** or [cybercrime.gov.in](https://www.cybercrime.gov.in).
